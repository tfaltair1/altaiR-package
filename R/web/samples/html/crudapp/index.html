Create Read Update Delete application

This demonstration shows how WPS Web can be used in a modern HTML5 & Javascript web application. The application comprises of a 'thick client' written in Javascript and executed by a user's web browser. This client renders the interface and makes a series of AJAX calls to the WPS Web Broker for language of SAS programs to be executed. These programs produce JSON text or GIF images which are then used to update the interface. The demonstration also shows how WPS Web can be used to create a " Create Read Update Delete" style application, which is popular pattern for modern web applications.
